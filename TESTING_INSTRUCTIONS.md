## 🔧 **If Issues Occur**

### **Generation Stuck?**
1. Refresh browser (F5)
2. Re-upload story
3. Check browser console (F12) for errors
4. Verify model is selected in AI Models tab

### **Server Errors?**
1. Check terminal window for error messages
2. Restart server: `npm run dev`
3. Clear browser cache

### **Model Not Working?**
1. Go to Downloads tab
2. Verify "Flux Minecraft Movie" shows "Installed"
3. Try reselecting in AI Models tab
